<h1 align="center">
  Poc Cache Redis
</h1>


Projeto simples para demonstrar o uso do cache no redis.

## Como Executar

- Executar o Redis:
```
docker-compose up
```
- Consultar Redis:
```
http://localhost:8081
```
- Consultar H2:
```
http://localhost:8080/h2
```
- Executar a aplicação Spring Boot
- Acessar aplicação em `http://localhost:8080`.
